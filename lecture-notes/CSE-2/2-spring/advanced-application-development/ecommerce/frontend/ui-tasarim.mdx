# Kullanıcı Arayüzü Tasarımı ve Stil Uygulaması

Bu bölümde, e-ticaret uygulamamızın kullanıcı arayüzü tasarımını ve stil uygulamasını detaylı olarak ele alacağız. Modern, duyarlı (responsive) ve kullanıcı dostu bir arayüz oluşturmak için Angular'ın stil özellikleri, CSS/SCSS ve Angular Material kütüphanesini nasıl kullanacağımızı inceleyeceğiz.

## Angular'da Stil Yaklaşımları

Angular, uygulamanızın stilini yönetmek için çeşitli yaklaşımlar sunar:

1. **Komponent Stilleri**: Her komponentin kendi stil dosyası olabilir ve bu stiller yalnızca o komponente uygulanır.
2. **Global Stiller**: Tüm uygulamaya uygulanan stil dosyaları.
3. **CSS Kütüphaneleri**: Bootstrap, Tailwind CSS gibi harici CSS kütüphaneleri.
4. **UI Komponent Kütüphaneleri**: Angular Material, PrimeNG, NG-Bootstrap gibi hazır UI komponent kütüphaneleri.

### Komponent Stilleri

Angular, her komponentin kendi stil dosyasına sahip olmasını destekler. Bu stiller, varsayılan olarak yalnızca o komponente uygulanır (view encapsulation):

```typescript
@Component({
  selector: 'app-product-card',
  templateUrl: './product-card.component.html',
  styleUrls: ['./product-card.component.scss']
})
export class ProductCardComponent { }
```

Bu örnekte, `product-card.component.scss` dosyasındaki stiller yalnızca `ProductCardComponent` komponentine uygulanır.

### View Encapsulation

Angular, stil kapsülleme (view encapsulation) için üç mod sunar:

1. **Emulated (Varsayılan)**: Angular, DOM'a eklenen benzersiz nitelikler kullanarak stilleri kapsüller.
2. **None**: Stiller global olarak uygulanır, kapsülleme yoktur.
3. **ShadowDom**: Tarayıcının Shadow DOM API'sini kullanarak stilleri kapsüller.

```typescript
@Component({
  selector: 'app-product-card',
  templateUrl: './product-card.component.html',
  styleUrls: ['./product-card.component.scss'],
  encapsulation: ViewEncapsulation.Emulated // Varsayılan
})
export class ProductCardComponent { }
```

### Global Stiller

Global stiller, `angular.json` dosyasında belirtilen stil dosyalarıdır ve tüm uygulamaya uygulanır:

```json
{
  "projects": {
    "e-commerce-app": {
      "architect": {
        "build": {
          "options": {
            "styles": [
              "src/styles.scss",
              "src/theme.scss"
            ]
          }
        }
      }
    }
  }
}
```

Bu örnekte, `styles.scss` ve `theme.scss` dosyaları global stil dosyalarıdır.

## SCSS Kullanımı

SCSS (Sassy CSS), CSS'in bir üst kümesidir ve değişkenler, iç içe kurallar, mixinler ve fonksiyonlar gibi güçlü özellikler sunar. Angular, SCSS'i destekler ve yeni bir Angular projesi oluşturduğunuzda SCSS'i seçebilirsiniz.

### SCSS Değişkenleri

SCSS değişkenleri, tekrar kullanılabilir değerler tanımlamanıza olanak tanır:

```scss
// src/styles.scss

// Renkler
$primary-color: #3f51b5;
$secondary-color: #ff4081;
$background-color: #f5f5f5;
$text-color: #333333;
$light-text-color: #ffffff;
$border-color: #e0e0e0;

// Yazı tipleri
$font-family: 'Roboto', sans-serif;
$font-size-small: 12px;
$font-size-medium: 14px;
$font-size-large: 16px;
$font-size-xlarge: 18px;
$font-size-xxlarge: 24px;

// Boşluklar
$spacing-xs: 4px;
$spacing-sm: 8px;
$spacing-md: 16px;
$spacing-lg: 24px;
$spacing-xl: 32px;

// Geçişler
$transition-speed: 0.3s;
$transition-function: ease-in-out;

// Gölgeler
$box-shadow: 0 2px 5px rgba(0, 0, 0, 0.1);
$box-shadow-hover: 0 4px 10px rgba(0, 0, 0, 0.15);

// Kırılma noktaları
$breakpoint-xs: 576px;
$breakpoint-sm: 768px;
$breakpoint-md: 992px;
$breakpoint-lg: 1200px;
```

### SCSS Mixinler

Mixinler, tekrar kullanılabilir stil blokları tanımlamanıza olanak tanır:

```scss
// src/styles.scss

// Duyarlı tasarım için mixin
@mixin respond-to($breakpoint) {
  @if $breakpoint == xs {
    @media (max-width: $breakpoint-xs) { @content; }
  } @else if $breakpoint == sm {
    @media (max-width: $breakpoint-sm) { @content; }
  } @else if $breakpoint == md {
    @media (max-width: $breakpoint-md) { @content; }
  } @else if $breakpoint == lg {
    @media (max-width: $breakpoint-lg) { @content; }
  }
}

// Flex kutusu için mixin
@mixin flex($direction: row, $justify: flex-start, $align: stretch, $wrap: nowrap) {
  display: flex;
  flex-direction: $direction;
  justify-content: $justify;
  align-items: $align;
  flex-wrap: $wrap;
}

// Geçiş için mixin
@mixin transition($property: all) {
  transition: $property $transition-speed $transition-function;
}

// Gölge için mixin
@mixin box-shadow($shadow: $box-shadow) {
  box-shadow: $shadow;
}

// Kırpılmış metin için mixin
@mixin truncate($lines: 1) {
  @if $lines == 1 {
    white-space: nowrap;
    overflow: hidden;
    text-overflow: ellipsis;
  } @else {
    display: -webkit-box;
    -webkit-line-clamp: $lines;
    -webkit-box-orient: vertical;
    overflow: hidden;
  }
}
```

## Duyarlı (Responsive) Tasarım

Duyarlı tasarım, uygulamanızın farklı ekran boyutlarına uyum sağlamasını sağlar. Angular uygulamanızda duyarlı tasarım uygulamak için çeşitli yaklaşımlar kullanabilirsiniz:

### CSS Media Queries

CSS media queries, farklı ekran boyutları için farklı stiller tanımlamanıza olanak tanır:

```scss
// src/app/product-list/product-list.component.scss

.product-grid {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  gap: $spacing-md;
  
  @media (max-width: $breakpoint-lg) {
    grid-template-columns: repeat(3, 1fr);
  }
  
  @media (max-width: $breakpoint-md) {
    grid-template-columns: repeat(2, 1fr);
  }
  
  @media (max-width: $breakpoint-sm) {
    grid-template-columns: 1fr;
  }
}
```

### Flexbox ve Grid

Flexbox ve Grid, duyarlı düzenler oluşturmak için güçlü CSS özellikleridir:

```scss
// src/app/home/home.component.scss

.featured-products {
  @include flex(row, space-between, stretch, wrap);
  
  .product-card {
    flex: 0 0 calc(25% - #{$spacing-md});
    margin-bottom: $spacing-md;
    
    @include respond-to(lg) {
      flex: 0 0 calc(33.333% - #{$spacing-md});
    }
    
    @include respond-to(md) {
      flex: 0 0 calc(50% - #{$spacing-md});
    }
    
    @include respond-to(sm) {
      flex: 0 0 100%;
    }
  }
}

.category-grid {
  display: grid;
  grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));
  gap: $spacing-md;
}
```

## Angular Material Kullanımı

Angular Material, Material Design rehberlerine dayalı yüksek kaliteli UI komponentleri sağlayan bir kütüphanedir. Angular Material'i kullanmak, modern ve tutarlı bir kullanıcı arayüzü oluşturmanıza yardımcı olur.

### Angular Material Kurulumu

Angular Material'i kurmak için:

```bash
ng add @angular/material
```

Bu komut, Angular Material'i kurar, tema seçmenizi sağlar ve gerekli modülleri `app.module.ts` dosyasına ekler.

// rating.component.ts

@Component({
  selector: 'app-rating',
  template: `
    <div class="rating">
      <mat-icon
        *ngFor="let star of stars; let i = index"
        [class.filled]="star"
        (click)="rate(i + 1)"
        (mouseenter)="hover(i + 1)"
        (mouseleave)="hover(0)"
      >
        {{ star ? 'star' : 'star_border' }}
      </mat-icon>
      <span *ngIf="showValue" class="rating-value">{{ value }}</span>

    .rating {
      display: flex;
      align-items: center;
    }

      cursor: pointer;
      color: #bdbdbd;
      transition: color 0.2s ease-in-out;
    }
    
    mat-icon.filled {
      color: #ffc107;
    }
    
    .rating-value {
      margin-left: 8px;
      font-weight: 500;
    }
  `]
})
export class RatingComponent {
  @Input() value = 0;
  @Input() max = 5;
  @Input() readonly = false;
  @Input() showValue = false;
  @Output() valueChange = new EventEmitter<number>();
  
  stars: boolean[] = [];
  hoverValue = 0;
  
  ngOnInit(): void {
    this.updateStars();
  }
  
  ngOnChanges(): void {
    this.updateStars();
  }
  
  rate(value: number): void {
    if (this.readonly) {
      return;
    }
    
    this.value = value;
    this.updateStars();
    this.valueChange.emit(this.value);
  }
  
  hover(value: number): void {
    if (this.readonly) {
      return;
    }
    
    this.hoverValue = value;
    this.updateStars();
  }
  
  private updateStars(): void {
    this.stars = Array(this.max).fill(false).map((_, i) => {
      return this.hoverValue > 0 ? i < this.hoverValue : i < this.value;
    });
  }
}

// theme.service.ts
import { Injectable } from '@angular/core';
import { BehaviorSubject } from 'rxjs';

@Injectable({
  providedIn: 'root'
})
export class ThemeService {
  private darkTheme = new BehaviorSubject<boolean>(false);
  isDarkTheme = this.darkTheme.asObservable();
  
  constructor() {
    // Kullanıcının tercihini localStorage'dan al
    const savedTheme = localStorage.getItem('darkTheme');
    if (savedTheme) {
      this.setDarkTheme(savedTheme === 'true');
    } else {
      // Kullanıcının sistem tercihini kontrol et
      const prefersDark = window.matchMedia('(prefers-color-scheme: dark)').matches;
      this.setDarkTheme(prefersDark);
    }
  }
  
  setDarkTheme(isDarkTheme: boolean): void {
    this.darkTheme.next(isDarkTheme);
    localStorage.setItem('darkTheme', isDarkTheme.toString());
    
    if (isDarkTheme) {
      document.body.classList.add('dark-theme');
    } else {
      document.body.classList.remove('dark-theme');
    }
  }
  
  toggleTheme(): void {
    this.setDarkTheme(!this.darkTheme.value);
  }
}

// product-list.component.ts
import { Component, OnInit, ViewChildren, QueryList, ElementRef } from '@angular/core';
import { ProductService } from '../../core/services/product.service';
import { Product } from '../../core/models/product.model';

@Component({
  selector: 'app-product-list',
  templateUrl: './product-list.component.html',
  styleUrls: ['./product-list.component.scss']
})
export class ProductListComponent implements OnInit {
  products: Product[] = [];
  loading = false;
  
  @ViewChildren('productCard') productCards: QueryList<ElementRef>;
  
  constructor(private productService: ProductService) { }
  
  ngOnInit(): void {
    this.loadProducts();
  }
  
  loadProducts(): void {
    this.loading = true;
    
    this.productService.getProducts().subscribe(
      response => {
        this.products = response.content;
        this.loading = false;
      },
      error => {
        console.error('Ürünler yüklenirken hata:', error);
        this.loading = false;
      }
    );
  }
  
  @HostListener('keydown', ['$event'])
  handleKeyDown(event: KeyboardEvent): void {
    if (event.key === 'ArrowRight' || event.key === 'ArrowLeft' || event.key === 'ArrowUp' || event.key === 'ArrowDown') {
      const cards = this.productCards.toArray();
      const currentIndex = cards.findIndex(card => card.nativeElement === document.activeElement || card.nativeElement.contains(document.activeElement));
      
      if (currentIndex !== -1) {
        let nextIndex = currentIndex;
        
        if (event.key === 'ArrowRight') {
          nextIndex = Math.min(currentIndex + 1, cards.length - 1);
        } else if (event.key === 'ArrowLeft') {
          nextIndex = Math.max(currentIndex - 1, 0);
        } else if (event.key === 'ArrowDown') {
          // Bir satır aşağı (örneğin, 4 sütunlu bir grid için)
          nextIndex = Math.min(currentIndex + 4, cards.length - 1);
        } else if (event.key === 'ArrowUp') {
          // Bir satır yukarı
          nextIndex = Math.max(currentIndex - 4, 0);
        }
        
        if (nextIndex !== currentIndex) {
          cards[nextIndex].nativeElement.focus();
          event.preventDefault();
        }
      }
    }
  }
}

## Sonuç

Bu bölümde, e-ticaret uygulamamızın kullanıcı arayüzü tasarımını ve stil uygulamasını detaylı olarak ele aldık. Angular'da stil yaklaşımları, SCSS kullanımı, duyarlı tasarım, Angular Material kullanımı, özel komponentler ve direktifler, animasyonlar, tema değiştirme, erişilebilirlik, performans optimizasyonu ve mobil uyumluluk konularını inceledik.

Modern, duyarlı ve kullanıcı dostu bir arayüz oluşturmak için Angular'ın stil özellikleri, CSS/SCSS ve Angular Material kütüphanesini nasıl kullanacağımızı öğrendik. Bu bilgiler, e-ticaret uygulamanızın kullanıcı deneyimini iyileştirmenize yardımcı olacaktır.

Bir sonraki bölümde, frontend ve backend entegrasyonunu ve uygulamanın dağıtımını ele alacağız.
