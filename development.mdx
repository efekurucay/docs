## Basic Markdown Features

### Text Formatting

You can use basic markdown formatting:
- **Bold text** using `**text**`
- *Italic text* using `*text*`
- ~~Strikethrough~~ using `~~text~~`
- `Code` using \`code\`

### Lists

Ordered lists:
1. First item
2. Second item
3. Third item

Unordered lists:
- Item one
- Item two
- Item three

### Code Blocks

```javascript
// With syntax highlighting
function hello() {
  console.log('Hello world!');
}
```

## Tips and Best Practices

1. Always provide descriptive titles and icons for cards and accordions
2. Use appropriate callouts for different types of information
3. Group related content using CardGroup or AccordionGroup
4. Include code examples when explaining technical features
5. Use steps for sequential instructions
6. Add navigation elements for better user experience
